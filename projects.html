Air Quality Analysis

Analysis of Global Plastic Pollution

This project analyzed the global distribution of plastic pollution. Through this project, I sought to find trends in the data that were indicative of higher rates of plastic pollution that is hard to recycle. Thekey findings were displayed in an interactive tableau dashboard.

Zillow Housing market analysis

Through this project, my partners and I were able to collect and analyze current housing data from Zillow. We were able to use webscraping to collect the data needed for the analysis. Upon the conclusion of this project, we were able to determine which states still had affordable housing prices.

NHANES Data Analysis

Upon analyzing the NHANES pre-pandemic data using R, my partners and I were able to create a random forest model that could accurately predict our unknown response variable. Through this project, we were able to investigate which predictors were indicative of high LDL cholesterol.

Zoo Project

Calculated the optimal solution to the travelling sales person problem, using a combination of TSP heuristics, and Prim's algorithm in order to implement brach and bounding. This program can find the minimum spanning tree and hamiltonian cycle connecting different points.

Euchre

This project involved using object oriented programming in C++ to simulate the card game euchre. The program can be customized to fit for different numbers of players. Users will play against a customizable number of computer players, who's strategy is based on an algorithm based on euchre strategies.
